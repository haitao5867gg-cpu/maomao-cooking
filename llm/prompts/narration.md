根据下面的结构化菜谱 JSON，为一条 <60 秒的竖屏动画短视频写旁白脚本。主角是一只可爱的猫厨师。

## 硬性规则
1. 所有用量、火候、时长**必须逐字来自 JSON**，一个数字都不能改、不能省、不能编。
2. 节奏结构（总时长 ≤ 58 秒，按 4.5 字/秒估算）：
   - 钩子（≤3s）：一句话让人想学这道菜
   - 食材卡（≤5s）：报出全部食材和用量
   - 步骤（每步 4-8s）：按 JSON steps 顺序，每步必须念出该步的用量/火候，并提及 visual_cue 作为"做到什么程度"的判断
   - 收尾（≤5s）：成品描述 + 一句 tips
3. 语气：轻快、口语化、像朋友教做菜；可以有一两处猫咪梗，但不挤占步骤信息。
4. 输出格式：JSON 数组，每个元素 `{{"scene": "hook|ingredients|step_N|outro", "narration": "...", "screen_text": "常驻字幕条内容（用量/火候）"}}`。只输出 JSON，不要其他文字。
